Windows 11 с чистого листа

Самая большая и затратная по времени задача, которую я делал в начале 2026 года. По ощущениям ушло 25+ часов, если включать в расчёт установку всех нужных мне программ. Решил накатить чистый Windows, чтобы очистить компьютерную память и собственный разум. За последние пять лет на компьютере скопилось очень много барахла, которое захотелось удалить всё и сразу. Создал чистый образ Windows 11 с помощью конфигуратора autoattend xml. Вырезал многие мусорные компоненты:

  • Copilot AI, Recall, Bing
  • Windows Defender, SmartScreen, Автоматическое обновление системы
  • Edge, Paint, Modern Notepad, XBox, OneDrive, и все стандартные программы
  • Онлайн аккаунт
  • Телеметрия

С конфигуратором autoattend пришлось повозиться, ибо раньше я им никогда не пользовался. Из стандартных программ оставил только Терминал, Часы и Калькулятор. Теперь у меня максимально чистый Windows 11 версии 25H2, на котором только свежие нужные мне программы 🧐 Впрочем, работа по приведению в порядок моих файлов ещё будет продолжаться. Позже собираюсь сортировать многочисленные файлы, которые хранятся у меня в облаке: фотографии, документы, архивы. Сколько на это уйдёт времени - боюсь представить. Ведь собирал я эти файлы почти всю жизнь, начиная с 2005 года. А сортировал я их примерно никогда 🤣😭♻️ Старое надо удалить, а нужное разложить по папкам 📂📂📂
