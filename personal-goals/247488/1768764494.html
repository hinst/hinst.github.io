-= AI =-

I keep generating different things for myself but I'm not sharing them. Some time ago, I learned about quantization for AI models. It appears that using the original model version is not always necessary. If you look online, you can find various versions compiled with a specific number of bits:

  • fp16, fp8
  • Q8, Q6, Q4, Q3, Q2

The number indicates the number of bits, and many models can be used in version Q4 with almost no quality loss, while this version is half the size of fp8 and four times smaller than fp16 üòè The only inconvenience is that you need to search for these versions on file dump sites like Huggingface and CivitAI; and hope that some kind user has collected and uploaded the desired build in a preferred format. From my experience, it's always better to get a powerful Q4 model than a weak one in Q8 or fp16. However, at levels Q3 and below, artifacts and hallucinations start to appear ü•µ
